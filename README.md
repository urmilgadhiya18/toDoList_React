## Installation & Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/urmilgadhiya18/toDoList_React.git
   ```
2. Navigate to the project directory:
   ```sh
   cd toDoList_React
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
